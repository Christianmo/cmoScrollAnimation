## CMO Scroll Animation

Basic JS scroll animation library.


### Install
```
npm i cmoscrollanimation
```

### Use

```javascript
import cmoScrollAnimation from 'cmoScrollAnimation';
```

```javascript
  cmoScrollAnimation(); 
```

```html
  <section class="animation">
    <p>Lorem ipsum.</p>
  </section>
```
